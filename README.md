# 🌍 지오게스 (GeoGuess)

구글 스트리트뷰로 랜덤한 장소를 보여주고, 지도에서 위치를 추측하면
**실제 위치와의 거리에 따라 점수**를 주는 GeoGuessr 스타일 모바일 웹 게임입니다.

- 🌐 **전세계 모드** — 스트리트뷰가 잘 깔린 전 세계 30개국 중 무작위 장소
- 📍 **특정 국가 모드** — 원하는 한 나라 안에서만 출제 (대한민국, 일본, 미국 등)
- 📱 모바일 우선 UI (전체 화면 스트리트뷰 + 접이식 추측 지도)
- 🏆 라운드당 최대 5,000점, GeoGuessr와 동일한 거리 기반 점수 공식

빌드 과정이 없는 순수 HTML/CSS/JavaScript라 정적 파일만 서빙하면 바로 실행됩니다.

## 실행 방법

```bash
# 저장소 폴더에서
python3 -m http.server 8000
# 또는
npx serve .
```

브라우저에서 `http://localhost:8000` 접속 → 처음 한 번 **Google Maps API 키**를 입력하면 시작됩니다.
(`http://localhost:8000/?key=발급키` 형태로 키를 바로 넘길 수도 있습니다.)

## Google Maps API 키 발급

1. [Google Cloud Console](https://console.cloud.google.com/) 에서 프로젝트 생성
2. **Maps JavaScript API** 사용 설정 (스트리트뷰 서비스 포함)
3. *API 및 서비스 → 사용자 인증 정보 → API 키* 생성
4. 게임 첫 화면에 키를 붙여넣기

> 키는 브라우저 `localStorage` 에만 저장되며 외부로 전송되지 않습니다.
> 운영 배포 시에는 Cloud Console에서 키에 **HTTP 리퍼러 제한**을 거는 것을 권장합니다.

## 게임 방법

1. 메뉴에서 모드 / 국가 / 라운드 수 / 이동 허용 여부 선택
2. 스트리트뷰를 둘러보며 단서(언어, 표지판, 풍경 등) 찾기
3. 지도(모바일은 "지도 열기")에서 추측 지점을 탭 → **여기로 추측!**
4. 실제 위치와의 거리·점수 확인 후 다음 라운드 진행
5. 모든 라운드가 끝나면 총점과 등급 확인

## 점수 공식

```
점수 = 5000 × e^(−10 × 오차거리(km) / 지도크기(km))
```

지도크기는 전세계 모드에서 14,916km(지구 기준), 국가 모드에서는 해당 국가 경계 상자의
대각선 길이를 사용합니다. 작은 나라일수록 더 정확히 맞혀야 높은 점수를 받습니다.

## 파일 구조

| 파일 | 설명 |
|------|------|
| `index.html` | 화면 구조 (키 입력 / 메뉴 / 게임 / 결과) |
| `styles.css` | 모바일 우선 다크 테마 스타일 |
| `countries.js` | 출제 국가 목록과 경계 상자 데이터 |
| `app.js` | 게임 로직 (스트리트뷰 탐색, 점수 계산, 지도 처리) |
