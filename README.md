# 🏎️ 네온 러시 (NEON RUSH)

순수 HTML/CSS/JavaScript + Canvas 로 만든 **모바일용 고속 레이싱 게임**입니다.
빌드 과정도, 서버도, API 키도, 외부 이미지도 필요 없습니다. `index.html` 만 열면 바로 실행됩니다.

## 바로 실행하기

- **GitHub Pages**: 저장소 *Settings → Pages* 에서 이 브랜치의 `/(root)` 를 소스로 지정하면
  `https://<사용자>.github.io/geoguess/` 에서 바로 플레이됩니다. (별도 빌드 불필요)
- **로컬**: `index.html` 파일을 브라우저로 더블클릭만 해도 동작합니다.
  (네트워크 요청이 전혀 없어 `file://` 로도 그대로 실행됩니다.)

## 게임 특징

- 🌌 신스웨이브 비주얼 — 석양 그라데이션, 줄무늬 태양, 패럴럭스 산맥, 네온 야자수
- 💨 강력한 속도감 — 네온 럼블 스트라이프, 화면 중심에서 뻗는 스피드 라인, 비네팅, 화면 흔들림
- 🛣️ 살아있는 도로 — 곡선·언덕·범프가 이어지는 pseudo-3D 트랙
- 🚗 피해야 하는 교통 차량 (부딪히면 급감속!)
- 🏁 90초 타임어택 — 제한시간 안에 최대 거리를 달리세요. 최고 기록은 브라우저에 저장됩니다.
- ⚡ 0 → 320 km/h 까지 4.5초, 자동 가속

## 조작

| | 모바일 | 키보드 |
|---|---|---|
| 좌/우 조향 | 화면 좌·우 `◀ ▶` 버튼 | `←` `→` (또는 `A` `D`) |
| 브레이크 | 가운데 `BRAKE` 버튼 | `↓` 또는 `Space` |
| 가속 | 자동 | 자동 |
| 재시작 | 오버레이 버튼 | `Enter` |

## 파일 구조

| 파일 | 설명 |
|------|------|
| `index.html` | 화면 구조 (캔버스 · HUD · 컨트롤 · 오버레이) |
| `style.css` | 모바일 우선 네온 테마 스타일 |
| `game.js` | 게임 엔진 (pseudo-3D 도로 렌더링, 물리, 교통, 점수) |
