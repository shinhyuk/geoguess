이 게임은 구글 지도(스트리트뷰) API 키가 필요합니다.

API 키는 어떻게 받나요?
  1. Google Cloud Console 접속 후 프로젝트 생성
  2. Maps JavaScript API와 Street View Static/Service를 사용 설정
  3. API 및 서비스 → 사용자 인증 정보 → API 키 생성
  4. 생성된 키를 위 입력란에 붙여넣기

키는 이 브라우저(localStorage)에만 저장되며 외부로 전송되지 않습니다.
